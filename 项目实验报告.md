## 一、项目概述

### 1.1 项目背景

随着教育信息化的发展，传统的纸质成绩管理方式已经无法满足现代教育管理的需求。本项目旨在开发一款基于JavaWeb技术栈的成绩管理系统，为学校、培训机构等教育场景提供高效、便捷的成绩管理解决方案，实现成绩录入、查询、统计分析等核心业务流程的数字化、智能化。

### 1.2 项目目标

- 建立完善的成绩管理体系，支持多角色用户操作
- 实现成绩的录入、修改、查询、统计等核心功能
- 提供智能化的AI学习助手，提升用户体验
- 确保系统的安全性、稳定性和可扩展性
- 打造简洁、友好的用户界面

### 1.3 应用场景

- **高等院校**：管理学生考试成绩、课程成绩
- **培训机构**：记录学员培训考核成绩
- **中小学校**：统计学生各科成绩和排名
- **在线教育平台**：追踪学习者的学习进度和考试表现

```java
// 计算平均分
public double calculateAverageScore(Integer examId) {
    List<Score> scores = scoreMapper.selectScoresByExamId(examId);
    if (scores == null || scores.isEmpty()) {
        return 0.0;
    }
    double sum = scores.stream()
            .mapToDouble(score -> score.getScore().doubleValue())
            .sum();
    return sum / scores.size();
}

// 获取最高分
public double getHighestScore(Integer examId) {
    List<Score> scores = scoreMapper.selectScoresByExamId(examId);
    return scores.stream()
            .mapToDouble(score -> score.getScore().doubleValue())
            .max()
            .orElse(0.0);
}
```

```java
@PostMapping("/login")
public ModelAndView login(@RequestParam("username") String username,
                         @RequestParam("password") String password,
                         HttpSession session) {
    ModelAndView modelAndView = new ModelAndView();
    
    User user = userService.login(username, password);
    if (user != null) {
        // 登录成功，保存用户信息到session
        session.setAttribute("user", user);
        modelAndView.setViewName("redirect:/main");
    } else {
        // 登录失败
        modelAndView.setViewName("login");
        modelAndView.addObject("error", "用户名或密码错误");
    }
    
    return modelAndView;
}
```

**后端实现（ScoreController.java）：**
```java
@PostMapping("/add")
public ModelAndView addScore(@RequestParam("studentId") String studentId,
                            @RequestParam("examId") Integer examId,
                            @RequestParam("score") BigDecimal scoreValue,
                            HttpSession session) {
    ModelAndView modelAndView = new ModelAndView();
    User currentUser = (User) session.getAttribute("user");
    
    Score score = new Score();
    score.setStudentId(studentId);
    score.setExamId(examId);
    score.setScore(scoreValue);
    score.setRecordedBy(currentUser.getUserId());
```

**后端实现（ScoreServiceImpl.java）：**
```java
@Override
public double calculateAverageScore(Integer examId) {
    List<Score> scores = scoreMapper.selectScoresByExamId(examId);
    if (scores == null || scores.isEmpty()) {
        return 0.0;
    }
    
    double sum = scores.stream()
            .mapToDouble(score -> score.getScore().doubleValue())
            .sum();
    return sum / scores.size();
}

@Override
public double getHighestScore(Integer examId) {
    List<Score> scores = scoreMapper.selectScoresByExamId(examId);
    return scores.stream()
            .mapToDouble(score -> score.getScore().doubleValue())
            .max()
            .orElse(0.0);
}

@Override
public double getLowestScore(Integer examId) {
    List<Score> scores = scoreMapper.selectScoresByExamId(examId);
    return scores.stream()
            .mapToDouble(score -> score.getScore().doubleValue())
            .min()
            .orElse(0.0);
}
```

```java
@PostMapping("/ask")
@ResponseBody
public Map<String, Object> askAI(@RequestParam("question") String question,
                                HttpSession session) {
    Map<String, Object> response = new HashMap<>();
    
    try {
        User currentUser = (User) session.getAttribute("user");
        if (currentUser == null) {
            response.put("status", "error");
            response.put("message", "请先登录");
            return response;
        }
        
        // 获取AI回复
        String aiResponse = getAIResponse(question);
        
        response.put("status", "success");
        response.put("answer", aiResponse);
        response.put("question", question);
        
    } catch (Exception e) {
        response.put("status", "error");
        response.put("message", "AI服务暂时不可用");
        e.printStackTrace();
    }
    
    return response;
}
```

### 10.4 常见问题FAQ

**Q1：如何修改端口号？**  
A1：修改`application.properties`中的`server.port=8082`为其他端口。

**Q2：忘记管理员密码怎么办？**  
A2：可以直接在数据库中修改：
```sql
UPDATE user SET password='admin' WHERE username='admin';
```

**Q3：AI助手无法使用怎么办？**  
A3：系统内置了智能回复算法，即使不配置AI API也能正常使用基本功能。

**Q4：如何添加新的测试数据？**  
A4：可以通过系统界面添加，或直接在数据库中插入数据。

**Q5：系统支持多少并发用户？**  
A5：当前配置支持100+并发用户，可通过调整连接池配置提升。
