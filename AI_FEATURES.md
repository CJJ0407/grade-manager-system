# 🤖 AI学习助手功能说明

## 功能概述

为成绩管理系统添加了免费的AI学习助手功能，帮助学生、教师和管理员答疑解惑。

## 🎯 主要功能

### 1. 智能问答
- 回答关于成绩查询的问题
- 解答考试相关信息
- 提供科目详情咨询
- 系统使用帮助

### 2. 用户界面
- 现代化的聊天界面
- 实时对话体验
- 快速问题按钮
- 响应式设计

### 3. 智能回复
- 基于关键词的智能匹配
- 上下文相关的回复
- 友好的交互体验
- 多轮对话支持

## 🚀 使用方法

### 访问方式
1. 登录系统后，在主页菜单中点击"🤖 AI学习助手"
2. 直接访问：http://localhost:8083/ai/chat
3. 查看帮助：http://localhost:8083/ai/help

### 支持的问题类型
- **成绩相关**：如何查看成绩、成绩说明等
- **考试相关**：考试时间、考试安排等
- **科目相关**：科目信息、课程详情等
- **系统使用**：功能操作、界面导航等
- **学习指导**：学习方法、学习建议等
